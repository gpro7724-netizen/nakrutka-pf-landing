# Скиллы для сайта по сеошке

Установлены из репозитория [AlexProTg/awesome-agent-skills](https://github.com/AlexProTg/awesome-agent-skills).

## Telegram / заявки

- **telegram-bot-builder** — логика бота и webhook’ов, если правим обработку заявок в Telegram.
- **telegram-automation** — автоматизация сценариев с заявками и уведомлениями.

## Дизайн и UX

- **frontend-design** — интерфейсы, визуальная идентичность, вёрстка лендинга.
- **ui-ux-designer** — UX/UI, паттерны интерфейсов, конверсионные блоки.

## Кодинг и API

- **api-patterns** — проектирование и вызов API (serverless-функции, отправка заявок).
- **error-handling-patterns** — обработка ошибок, логирование, сообщения пользователю.
- **python-pro** — для бэкенд‑части и вспомогательных скриптов (если появятся).

В каждой папке скилла должен лежать файл `SKILL.md` из `awesome-agent-skills`. Если какой‑то скилл нужен — скопируй его из `awesome-agent-skills` в `.cursor/skills/ИМЯ-СКИЛЛА/`.
