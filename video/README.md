# Видеофон для Hero — кузница

Фон блока Hero должен показывать **видео кузницы** (ковка металла, искры, наковальня): мы «куём» сайты клиентов. В видео не обязательно наличие надписи «ваш сайт» — она выведена поверх видео в интерфейсе.

В папке должен лежать файл **hero-bg.mp4**. Замените текущий ролик на подходящее видео кузницы/ковки.

## Где взять бесплатное видео кузницы

1. **Mixkit** (скачать MP4, сохранить как `hero-bg.mp4`):
   - [Welding a metal bar](https://mixkit.co/free-stock-video/welding-a-metal-bar-32817/) — искры, металл
   - [Welding in a metal factory](https://mixkit.co/free-stock-video/welding-in-a-metal-factory-20910/) — заводская среда
   - Поиск по запросам: «blacksmith», «forge», «anvil», «metal», «spark»

2. **Pexels** — https://www.pexels.com/search/videos/blacksmith/ или «forge», «metal forging»

3. **Pixabay** — https://pixabay.com/videos/search/forge/

Скачайте MP4 (H.264), лучше 1920×1080. Положите файл в эту папку как **hero-bg.mp4** — фон Hero будет показывать кузницу, поверх него отображается текст «ваш сайт».
